## Generador de proyectos NPM

This CLI Generator use to generate React different projects in very easy steps.

# Requirements
* Node >= 10.15.3
* NPM >= 6.0.0

# Install
```npm install -g React-Projects-2022/react-cli-project-generator```
# Usage
Execute to use
```react-cli-anartz```
